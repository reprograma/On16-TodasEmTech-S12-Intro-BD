## Arquitetura MVC

Mais uma vez, todo projeto foi montado com arquitetura MVC, com as pastas:

- [x] controllers - para armazenar a lógica de controle da nossa api
- [x] models - para armazenar os nosso modelos (ex: nossas músicas)
